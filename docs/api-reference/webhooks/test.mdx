---
title: 'Test Webhook'
api: 'POST /api/webhooks/{id}/test'
---

Enqueue a sample event to the destination to exercise the endpoint. It sends a real, signed
message of a concrete type the destination is subscribed to (the first of its `eventTypes`,
defaulting to `sandbox.created`), so it respects the destination's filters. Delivery is
**asynchronous** — this returns once the message is accepted; check
[deliveries](/api-reference/webhooks/deliveries-list) for the outcome. See [Webhooks](/sandboxes/webhooks).

<ParamField path="id" type="string" required>
  Destination ID (`whk_…`).
</ParamField>

<ResponseExample>
```json 200
{
  "ok": true,
  "eventType": "sandbox.created",
  "messageId": "msg_2aB…"
}
```
</ResponseExample>

`messageId` is the delivery's message id — find its attempt under [deliveries](/api-reference/webhooks/deliveries-list).
