---
title: 'Test Webhook'
api: 'POST /api/webhooks/{id}/test'
---

Send a synthetic `sandbox.test` event to the destination and return the result inline. It bypasses the `eventTypes` filter, is signed like a real delivery, is not retried, and does not appear in delivery history. See [Webhooks](/sandboxes/webhooks).

<ParamField path="id" type="string" required>
  Destination ID (`whk_…`).
</ParamField>

<ResponseExample>
```json 200
{
  "delivered": true,
  "responseCode": 200
}
```
</ResponseExample>

`delivered` is `false` with an `error` string when the endpoint didn't return `2xx`.
