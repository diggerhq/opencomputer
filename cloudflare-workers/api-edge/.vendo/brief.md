OpenComputer is a cloud sandbox platform for developers and teams running code, browsers, agents, checkpoints, previews, templates, credentials, and usage billing across regional cells. The Cloudflare edge app authenticates users and API keys, serves the dashboard SPA, enforces org policy, routes sandbox and checkpoint operations to the owning cell, manages dashboard data in D1, and handles billing, webhooks, secret stores, snapshots, templates, and internal reconciliation endpoints. The agent should help operators and users inspect platform state, manage sandbox resources, understand usage and billing, and make guarded changes through the host's own APIs.
